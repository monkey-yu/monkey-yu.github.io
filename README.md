
# 欢迎来到monkey-yu Blog
## 环境<br/>
1. [安装Ruby运行环境](http://www.cnblogs.com/daguo/p/4097263.html)（需安装ruby >=2.3的版本）
2. 安装Jekyll以及运行。
```
gem install jekyll
gem install jekyll-paginate
jekyll serve
```
服务运行地址：http://127.0.0.1:4000/
